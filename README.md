# 클래스 기타

## 사용 방법

1. Visual Studio에서 `Chapter30-ClassMisc.slnx`를 엽니다
2. 솔루션 탐색기에서 프로젝트를 선택하고 `README.md`를 읽습니다
3. `Program.cs`에 코드를 작성합니다
4. 해당 프로젝트를 시작 프로젝트로 설정하고 실행합니다 (`Ctrl+F5`)

## 코딩 연습

`CodingPractice` 프로젝트의 README를 따라 교안 예제 코드를 직접 작성하고 실행합니다.

| 프로젝트 | 내용 |
|----------|------|
| CodingPractice | 교안 코드 따라치기 (16개 과제) |

## 연습문제

각 프로젝트의 README에 있는 문제를 읽고 코드를 작성합니다.

| 프로젝트 | 문제 |
|----------|------|
| PartialRecipe | 부분 클래스로 레시피 카드 만들기 |
| StaticUtils | 정적 유틸리티 클래스 구현 |
| QueryBuilder | 메서드 체이닝 쿼리 빌더 구현 |
| ImmutableMoney | 불변 형식 Money 클래스 구현 |
