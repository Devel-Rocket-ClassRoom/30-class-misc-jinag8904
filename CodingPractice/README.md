# 코딩 연습 - 클래스 기타

교안의 예제 코드를 직접 따라 작성하고 실행하는 프로젝트입니다.

> 각 과제의 코드를 `Program.cs`에 작성하세요.
> 이전 과제의 코드가 남아있으면 변수 이름이 겹쳐서 오류가 발생할 수 있습니다. 이전 과제의 코드를 지우거나 주석 처리한 후 다음 과제를 진행하세요.

---

## 과제 1: 부분 클래스 기본 문법

같은 파일 내에서 부분 클래스를 사용하여 `Person` 클래스를 구현하세요.

- 첫 번째 `partial class Person`에 `Name`, `Age` 속성을 정의합니다
- 두 번째 `partial class Person`에 `Print()` 메서드를 정의합니다
- `Person` 인스턴스를 생성하고 이름과 나이를 설정한 후 출력합니다

**실행 결과**

```
홍길동: 25
```

## 과제 2: 부분 클래스 사용 예제

`Hello` 클래스를 부분 클래스로 나누어 구현하세요.

- 첫 번째 부분에 `Hi()` 메서드를 정의합니다 ("안녕하세요!" 출력)
- 두 번째 부분에 `Bye()` 메서드를 정의합니다 ("안녕히 가세요!" 출력)
- `Hello` 인스턴스를 생성하고 두 메서드를 호출합니다

**실행 결과**

```
안녕하세요!
안녕히 가세요!
```

## 과제 4: 유틸리티 클래스 만들기

문자열 처리 유틸리티인 `StringHelper` 정적 클래스를 구현하세요.

```
olleH
```

## 과제 7: 메서드 체이닝 직접 구현하기

`MessageBuilder` 클래스를 구현하여 메서드 체이닝을 직접 만들어 보세요.

- `AddText(string text)`: 텍스트 추가 후 `this` 반환
- `AddSpace()`: 공백 추가 후 `this` 반환
- `AddNewLine()`: 줄바꿈 추가 후 `this` 반환
- `Build()`: 최종 문자열 반환
- 메서드 체이닝으로 메시지를 조합하여 출력합니다

**실행 결과**

```
안녕하세요!
반갑습니다. 좋은 하루 되세요.
```

## 과제 8: 불변 객체와 메서드 체이닝

새 객체를 반환하는 방식의 메서드 체이닝을 구현하세요.

- `Point` 클래스에 `readonly` 필드 `X`, `Y`를 정의합니다
- `MoveBy(int dx, int dy)` 메서드는 새 `Point` 객체를 반환합니다
- `ToString()`을 오버라이드하여 `"(X, Y)"` 형식으로 출력합니다
- 메서드 체이닝으로 `(0,0)` → `(10,10)` → `(30,30)` → `(60,60)` 이동합니다

**실행 결과**

```
(60, 60)
```

## 과제 10: 가변 형식

값을 직접 변경할 수 있는 가변 형식 `MutableCircle` 클래스를 구현하세요.

- `Radius` 속성을 `get; set;`으로 정의합니다
- 생성자에서 초기 반지름을 설정합니다
- 반지름을 10으로 생성한 후 출력하고, 20으로 변경한 후 다시 출력합니다

**실행 결과**

```
반지름: 10
반지름: 20
```

---

## 과제 11: 불변 형식

상태를 변경할 수 없는 불변 형식 `ImmutableCircle` 클래스를 구현하세요.

- `Radius` 속성을 `get`만으로 정의합니다 (읽기 전용)
- `WithRadius(int newRadius)` 메서드는 새 `ImmutableCircle` 객체를 반환합니다
- 반지름 10으로 생성하고, `WithRadius(20)`으로 새 원을 만들어 원본과 비교합니다

**실행 결과**

```
원본 반지름: 10
새 원 반지름: 20
원본 반지름: 10
```

## 과제 14: 종합 - 부분 클래스로 게임 캐릭터 구현

`Character` 클래스를 부분 클래스로 나누어 구현하세요.

- 첫 번째 부분에 `_name`, `_health`, `_level` 필드와 속성, 생성자를 정의합니다

**실행 결과**

```
용사이(가) 10만큼 회복함. 현재 체력: 80
```

## 과제 15: 종합 - 정적 클래스로 유틸리티 만들기

게임용 유틸리티 정적 클래스 `GameHelper`를 구현하세요.

- `CalculateDamage(int baseDamage, int level)`: 기본 데미지 + 레벨 * 5 반환
- `IsAlive(int health)`: 체력이 0보다 큰지 반환
- `GetHealthStatus(int health, int maxHealth)`: 비율에 따라 "양호"/"주의"/"위험"/"사망" 반환
- 각 메서드를 테스트합니다

**실행 결과**

```
계산된 데미지: 35
생존 여부: True
체력 상태: 주의
```

## 과제 16: 종합 - 불변 형식과 메서드 체이닝 결합

`Vector2D` 불변 클래스를 구현하고 메서드 체이닝으로 벡터 연산을 수행하세요.

- `readonly` 필드 `X`, `Y`를 정의합니다
- `Add(double x, double y)`: 새 `Vector2D` 반환
- `Multiply(double scalar)`: 새 `Vector2D` 반환
- `ToString()`: `"(X, Y)"` 형식으로 반환
- `(1,1)` → Add(2,3) → Multiply(2) → Add(-1,-2) 체이닝으로 결과를 출력합니다

**실행 결과**

```
결과 벡터: (5, 6)
```
